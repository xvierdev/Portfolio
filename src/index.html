Projetos

API FATEC - SCRUM 360

Desenvolvimento de uma aplicação Web para ensinar a metodolgia Scrum aos funcionários de uma empresa fictícia.

Elaboração dos conteúdos Scrum com exemplos práticos do cotidiano, desenvolvemos uma página Web separada em quatro módulos para os papéis da metodologia ágil, ao Final de cada módulo implementamos um questionário de avaliação parcial para que o usuário possa acompanhar seu próprio desempenho.
Utilizamos o framework tailwind para estilização da página, também utilizamos além do HTML, CSS e Javascript, um pouco de Python utilizando o microframework Flask para executar a aplicação em uma máquina virtual na AWS.
